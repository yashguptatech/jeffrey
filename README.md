# jeffrey

Cinematic, satirical personal-brand parody website built with Next.js.

## Stack

- Next.js (App Router)
- Tailwind CSS
- Framer Motion
- GSAP
- Three.js + React Three Fiber
- Lenis smooth scrolling

## Run locally

```bash
npm install
npm run dev
```

Then open <http://localhost:3000>.

## Build + lint

```bash
npm run lint
npm run build
```
